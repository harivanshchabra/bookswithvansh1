- Responsive design that works on all devices

- Service cards with hover effects
- Testimonials section with 5-star ratings
- Newsletter signup section
- Final CTA section
- Footer with contact information

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   \`\`\`bash
   git clone <repository-url>
   cd my-react-app
   \`\`\`

2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

3. Start the development server:
   \`\`\`bash
   npm run dev
   \`\`\`

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

\`\`\`bash
npm run build
\`\`\`

### Preview Production Build

\`\`\`bash
npm run preview
\`\`\`

## Customization

### Icons

Replace the icon images in the `public/icons/` directory with your own custom icons.

### Colors

The primary accent color is `#6C63FF` (purple). To change it, update all occurrences in the component files.

### Content

Edit the text content in each component file to customize copy, headings, and descriptions.
